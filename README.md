# Advanced / Dynamic Form Project

## Description
This repository contains a dynamic multi-step or conditional form built using **Angular / JavaScript**. The form adapts to user inputs, showing or hiding fields based on previous answers, and performs validation dynamically.

## Features
- Multi-step / conditional form workflow  
- Dynamic field generation based on user responses  
- Real-time input validation  
- Smooth and interactive user experience  

## Tech Stack
- JavaScript & / or Angular  
- HTML5  
- CSS3  

## Installation
```bash
git clone https://github.com/purisha/form.git
cd form
npx http-server .

Usage
	•	Start the form and follow steps
	•	Fields appear dynamically based on your input
	•	Validation ensures all required data is correct
